For remembering Javascript and jQuery terminology

Javascript

An object-oriented computer programming language used to create interactive effects within web browsers.

Operators

A symbol that's used to perform an operation, most often (but not always) arithmetic.

Variable

Anything that can vary. Variables hold data values that can be changed anytime.

JS Variable Naming Conventions

camelCase for identifier names (variables and functions). All names start with a letter. Sometimes global variables are written in UPPERCASE.

Function

A function is a JavaScript procedureâ€”a set of statements that performs a task or calculates a value. To use a function, you must define it somewhere in the scope from which you wish to call it.

Method

JavaScript methods are actions that can be performed on objects. A JavaScript method is a property containing a function definition. For example, a Window object would have methods such as open and close, while its state (whether it is opened or closed) would be a property.

Argument

Values passed to a function when it is invoked e.g. subtractFunction(3,2)

Parameter

Variables listed as part of the function defintion e.g., going on the example above, subtractFunction(number1, number2)

Return

A statement that stops the execution of a funtion and returns a value from that function to the caller.

Chaining Methods

A technique that can be used to simplify code in scenarios that involved calling multiple functions on the same object consecutively. Cleans up your code!

Strings

This means words not numbers. A global object & constructor for strings of a sequence of characters.

Boolean

A binary variable, having two possibel values called "true" or "false". Even though these are text, they're not considered strings.

Undefined

A global property that represents the primitive value undefined. One of Javascript's primitive types. This is what gets returned when a variable is declared but undefined.

NaN

Represents the Not-a-Number value. This property indicates that a value is not a legal number.
